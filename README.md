# mlsh-vocabulary
## 網頁介紹

這是一個包含單字卡和測驗模式的英語學習網站。

### 單字卡功能

- 顯示英文單字及其中文解釋
- 左上角索引可調整單字卡順序
- 右下角方向鍵可切換單字卡
- 左下角圖示可朗讀單字或顯示中文解釋

![單字卡示意圖](https://hackmd.io/_uploads/HJkB1CoJR.png)

### 測驗模式

提供四種測驗方式:

1. **看英文選中文**: 根據英文單字選擇中文解釋
2. **看中文選英文**: 根據中文解釋選擇英文單字
3. **文意字彙**: 根據提示詞選擇最符合文意的單字
4. **地獄版文意字彙**: 更具挑戰性的文意字彙測驗

選擇章節後即可開始測驗。

![測驗模式示意圖](https://hackmd.io/_uploads/ByFtyCsJ0.png)
